## üö® Red Flags (Scam Warnings)

**Be careful if buyer**:
- Asks to transact outside Flippa escrow
- Wants to pay with cryptocurrency only
- Asks for code before payment
- Pressures you to "act now"
- Offers way above asking price (too good to be true)
- Asks for your personal financial info

**Always use**:
- Flippa escrow OR
- Escrow.com (for large transactions)

**Never**:
- Share code before payment clears
- Send code via email before escrow releases
- Accept payment via Western Union, gift cards, etc.
